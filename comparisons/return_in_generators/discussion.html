What happens when you return in a generator?

In Node, the next() method returns an object with properties value and done, as usual, where value is what was returned and done is true.

In Python, return in a generator raises a StopIteration, and the exception's value attribute is set to the return value.

Alike in that return finishes a generator. Unalike in how the return value is extracted: out the front door, in the usual way, or smuggled out the back.
